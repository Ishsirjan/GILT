Graded Incomplete Letter Test (GILT)

This digital tool simulates the GILT task used to assess cortical visual function â€” particularly in individuals with dementia-related visual impairment. It is based on the scientific framework presented in:

Yong, K. X. X., et al. (2024). A digital test for cortical visual dysfunction: Graded Incomplete Letters (GILT).
Behavior Research Methods.

This test presents degraded letter images across multiple completeness levels (e.g., 100% to 1.58%). The assessment follows stopping rules and threshold calculations derived from the original clinical study.
